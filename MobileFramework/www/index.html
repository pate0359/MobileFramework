Welcome!

This is the sample cordova application with two basic functionality GEOLOCATION API and CONTACT API.


Geolocation API

Application will fetch user's current location, and will get the static map image using google static map API.It uses canvas to display image. Current location marker will be displayed on static map image. Make sure you allow permission to access your location, when application ask you, or application can not get your location neither static map image.

Contact API

Application will get user's phone contacts using Contact API and display as a list. Make sure you allow permission to access your contacts information, when application ask you, or application can not get your contacts list.
